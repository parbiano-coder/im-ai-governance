## 주요 기능
- AI 서비스 등록 및 위험등급(고/중/저위험) 자동 산정
- 문서 업로드 → Claude AI 자동 위험평가 → Word 보고서 생성
- 기획·개발·운영 3단계 위험평가 체크리스트 자동화
- AI기본법·금융분야 AI 안내서 기준 보고서 자동 작성
- Claude AI 거버넌스 어시스턴트 실시간 질의응답

## 사용 방법
설정 메뉴에서 Anthropic API 키(`sk-ant-...`) 입력 후 모든 AI 기능 활성화됩니다.
